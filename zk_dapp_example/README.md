## ZK Example Dapp
2つの入力を受け取り、以下をzkpで保証する
- それらが両方とも0から5の間であること
- それらが両方とも互いに等しくないこと
2つの値を掛け算して出力する

## Getting Started
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
